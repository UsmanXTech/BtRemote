# Tech Stack - BtRemote

## Architecture
- **Clean Architecture**: Divided into presentation, domain, and data layers.
- **Dependency Injection**: Koin.
- **Navigation**: Jetpack Compose Navigation.

## Frontend
- **Framework**: Jetpack Compose.
- **UI Components**: Material 3.
- **State Management**: StateFlow, ViewModel, collectAsStateWithLifecycle.
- **App Widgets**: Android Glance.

## Backend / System Integration
- **Platform**: Android SDK (Kotlin).
- **Communication**: Bluetooth HID Device Profile.
- **Sensors**: Android SensorManager (Gyroscope).
- **Data Storage**: Android DataStore (Preferences).
- **Concurrency**: Kotlin Coroutines & Flow.

## Tools & Libraries
- **Gradle**: Kotlin DSL (build.gradle.kts).
- **Linter/Static Analysis**: (To be verified, likely Kotlin-standard).
- **Version Control**: Git.
