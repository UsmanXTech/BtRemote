# Product Definition - BtRemote

## Overview
BtRemote is an Android application that transforms your smartphone into a versatile Bluetooth HID (Human Interface Device) remote control. It allows users to control other devices (e.g., computers, smart TVs, or other smartphones) as a keyboard, mouse, or media controller.

## Core Features
- **Bluetooth HID Keyboard**: Send text or key commands over Bluetooth.
- **Bluetooth HID Mouse**: Use the screen as a trackpad or utilize the gyroscope as an "air mouse".
- **Media Controls**: Dedicated interface for playback, volume, and navigation.
- **Device Discovery & Management**: Scan and connect to target devices.
- **Widget Integration**: Control media playback directly from the home screen using Android Glance.
- **Service-Based Operation**: Continues running in the background for consistent control.

## Target Audience
- Users looking to control their media centers from a distance.
- People who need a portable backup input device.
- Users seeking an intuitive "air mouse" experience.
